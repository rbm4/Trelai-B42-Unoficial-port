# Trelai Map 4x4 - Unofficial B42 Port

Presented by: Hypothetic

This repository contains an unofficial Project Zomboid Build 42 port of the Trelai 4x4 map.

## Important Disclaimer

- I am not the original author of this map.
- This is an unofficial port.
- If the original author requests removal, open an issue or leave a comment and the port will be removed.

## Port Details

- The map was reverse engineered to make it compatible from B41 to B42.
- It is provided as-is, with no support of any kind.
- The map works mostly fine, but this is still an unofficial port while the original has not been ported.

## Repository Purpose

This repo exists so community members can:

- Track updates and fixes.
- Report issues.
- Contribute improvements through pull requests.

## How To Contribute

1. Fork this repository.
2. Create a branch for your change.
3. Make your edits and test in-game.
4. Open a pull request with a clear summary of what changed.

## Pull Request Guidelines

- Keep changes focused and small when possible.
- Describe affected files and gameplay impact.
- Include steps to reproduce/fix when relevant.
- Be respectful of the original map ownership and this repository's unofficial status.
